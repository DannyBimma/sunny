# sunny

Terminal simulation of the tech on-board the Thousand Sunny pirate ship, from the One Piece manga and anime.

Usage
- Build: `./build.sh` (requires Xcode CLT)
- Run without args to show current location and help: `./run-sunny.sh`
- Systems:
  - `./run-sunny.sh coupe 2`     (COUP DE BURST: move north 2 km)
  - `./run-sunny.sh chicken 3`   (CHICKEN VOYAGE: move south 3 km)
  - `./run-sunny.sh rabbit 10`   (RABBIT SCREW: move north 10 km)
  - `./run-sunny.sh cannon 5`    (GAON CANNON: power level 5)
  - `./run-sunny.sh dock 3`      (SOLDIER DOCK: describe + launch channel 3)
  - `./run-sunny.sh garden 12`   (USOPP'S GARDEN: show Pop Green #12)
  - `./run-sunny.sh mikan 1`     (NAMI’S GARDEN: access warning)
  - `./run-sunny.sh fluer 1`     (ROBIN’S FLOWERS: access warning)
